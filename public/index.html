The Web Ring Revolution

What are you waiting for?

HTML, or the hypertext markup language, was originally designed to enrich text with certain attributes that would embellish its display within a web browser.

Nowadays, HTML has been enhanced by numerous technologies including CSS and JavaScript.

HTML encapsulates the structure of a document. Using hierarchical tags, you construct a tree of nodes (with optional text content) that is exposed to JavaScript as the Document Object Model (DOM).

JavaScript is the coding language of the web. When you embed JavaScript in your page via a <script> tag, you add the ability to execute logic as well as utilize an ever-expanding catalog of web APIs

CSS is the presentation layer of the modern web. It stands for "Cascading Style Sheets". Styles specified for a parent element will "cascade" (propagate) to its children within the DOM tree. CSS also enables selectors based on an element's attributes (most commonly id and class) that can be used to further subdivide behavior.

Using all or just one of these technologies, you — a person with a text editor — can publish content to the web or create an interactive experience.
